# Rainfall

A mobile-first 3D roguelike in the spirit of Risk of Rain 2.

The game lives in [`./game`](./game). See [`game/README.md`](./game/README.md)
for setup, phone-testing instructions (LAN Wi-Fi, ngrok, Capacitor native wrap),
and an overview of the code structure.

Quickstart:

```bash
cd game
npm install
npm run dev
```

Then open the printed Network URL on your phone (same Wi-Fi as your computer).
